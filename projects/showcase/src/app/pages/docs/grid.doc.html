The Grid is the flagship of cerious-widgets — a virtualized data grid with sorting, filtering, grouping, pinning, selection, inline editing, custom cell templates, pagination and Excel export. Behaviour is configured through gridOptions, columns opt into capabilities via their ColumnDef, and larger features (filtering, export, column menu…) are opt-in plugins you register once at bootstrap. The Grid's heavier features are plugins — self-contained units built on the shared WidgetPlugin contract and wired up by the PluginManagerService. Register the ones you need once at bootstrap; each page then turns on what it wants through pluginOptions (a plugin is inert until enabled). This keeps the core grid small and lets you tree-shake what you don't use.
